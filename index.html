jQuery Spoiler

Minimal, lightweight jQuery spoiler plugin

Just how minimal and lightweight?

  • Only one CSS requirement
  • Fully customizable CSS
  • Small size (< 2KB minified)

Usage

    
<script>
  // Basic usage
  $(".spoiler").spoiler();

  // Enabling some options
  $(".spoiler").spoiler({
    buttonName: "Hidden",
    triggerEvents: true
  });
</script>
